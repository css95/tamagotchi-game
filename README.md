# Pet Garden
A browser-based Tamagotchi-style game built as the final project in my JavaScript 2 course.

🔗 **Live Demo:** [View Website](https://css95.github.io/tamagotchi-game)

## Features
- Manage up to 4 pets simultaneously
- Energy, fullness and happiness stats that decay over time
- Auto-decay timer running in the background
- Random pet name generation via external API
- OOP architecture with a Pet class

## Tech Stack
- HTML
- CSS
- JavaScript
  
## Run Locally
```bash
git clone https://github.com/css95/tamagotchi-game.git
cd tamagotchi-game
//open index.html
```
